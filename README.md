# 🚀 AlgoVisualizer

AlgoVisualizer is an interactive web application that visually demonstrates how popular sorting and searching algorithms work in real time. The project helps students and developers understand algorithm behavior through dynamic animations, comparisons, swaps, and traversals directly in the browser.

🔗 **Live Demo:**  
https://bindusri2605.github.io/algo-visualizer/

---

## 📌 Overview

Understanding algorithms through theory alone can be difficult. AlgoVisualizer simplifies this process by providing a visual and interactive learning experience where users can observe each algorithm step-by-step.

The project is built using core web technologies without external frameworks, making it lightweight, beginner-friendly, and easy to understand.

Users can:
- Visualize sorting and searching algorithms interactively
- Observe comparisons and swaps in real time
- Adjust animation speed and array size
- Learn algorithm flow intuitively through animations

---

## ✨ Features

### 🔹 Sorting Algorithms
The application visualizes the working of:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

Each array element is represented using animated vertical bars whose heights correspond to their values.

---

### 🔹 Searching Algorithms

- Linear Search
- Binary Search

Elements are dynamically highlighted during traversal to visually represent the searching process.

---

### 🎛️ Interactive Controls

- Adjustable array size
- Adjustable animation speed
- Start and reset controls
- Responsive and clean user interface
- Real-time algorithm execution visualization

## ⚙️ How It Works

1. Select a sorting or searching algorithm.
2. Configure array size and animation speed.
3. Start the visualization.
4. Watch comparisons, swaps, and traversals animate step-by-step.

The project uses JavaScript DOM manipulation and asynchronous execution to dynamically update visual elements during algorithm execution.
